# KnowledgeExtract

This project uses data from [nlpdata/dialogre](https://github.com/nlpdata/dialogre).

## Setup

1. **Create a virtual environment:**
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

2. **Install dependencies:**
    ```bash
    pip3 install -r requirements.txt
    ```
